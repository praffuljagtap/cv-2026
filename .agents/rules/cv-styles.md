- Two-Column Tabular Layout: Use a display: grid or flex layout where the left column (dates) is roughly 25-30% width and the right column (content) takes the rest.

- Reverse-Chronological Order: Explicitly set the sorting rule so the most recent experience and education appear at the top.

- Date Precision: Format all dates as MM/YYYY - MM/YYYY or MM/YYYY - Present.

- Typography: Set the base font to a clean Sans-Serif (e.g., 'Inter', 'Arial', or 'Helvetica') with a size between 10pt and 11pt for readability.

- Minimalist Accents: Use a single accent color (usually a dark navy or charcoal) for section headers only; avoid gradients or decorative icons.

- The "Signature" Block: Include a footer rule that creates a space at the bottom-left for [City], [Current Date] and a placeholder for a physical/digital signature.

- Professional Headshot: Define a CSS class for a 1:1 or 4:3 ratio photo placeholder in the top right corner with a subtle border or no styling at all.

- Standard Margins: Maintain a minimum margin of 1.5cm to 2.0cm on all sides to ensure it looks balanced when printed or saved as a PDF.
